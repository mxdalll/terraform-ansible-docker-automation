# 🚀 Infrastructure Automation using Terraform, Ansible & Docker

A production-style DevOps project that provisions AWS infrastructure using Terraform and automates server configuration and application deployment using Ansible and Docker.

---

## 📖 Project Overview

This project demonstrates an end-to-end infrastructure automation workflow.

Instead of manually creating and configuring cloud resources, the complete deployment is automated using Infrastructure as Code (Terraform) and Configuration Management (Ansible).

The application is containerized using Docker and deployed on an Amazon EC2 instance.

## 🚀 Deployment Workflow

1. Configure AWS credentials
2. Run Terraform to provision infrastructure
3. Configure EC2 using Ansible
4. Deploy Docker container
5. Access application using EC2 Public IP
